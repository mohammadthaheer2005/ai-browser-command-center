## üèóÔ∏è Technical Architecture
The system is built on a modular "Shell-and-Core" architecture for maximum security and scalability.

*   **Logic Engine:** Unified LLM interface (Gemini 1.5 Pro / GPT-4o) using `browser-use`.
*   **Orchestration:** Async background threads to prevent GUI locking.
*   **Persistence:** Localized browser profiling for one-time logins.
*   **Security Layer:** Encrypted integrity checks to prevent unauthorized standalone execution.

## üöÄ Getting Started

### Prerequisites
*   Windows 10/11
*   Python 3.9 or higher
*   A valid Google Gemini or OpenAI API Key

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/mohammadthaheer2005/ai-browser-command-center.git
   cd ai-browser-command-center
   ```

2. **Setup Dependencies**
   ```bash
   pip install -r requirements.txt
   playwright install chromium
   ```

3. **Activation**
   > **Note:** Due to the proprietary nature of the core AI orchestration, this project requires a local security configuration file (`internal_sys_config.dat`) to activate the automation engine. For enterprise access or research collaboration, please contact the repository owner.

4. **Launch**
   ```bash
   python AgentApp.py
   ```
